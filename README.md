# 📱 Overview

UrbanCart is a full-stack ecommerce mobile application that provides a seamless shopping experience for customers while allowing sellers to manage their products efficiently.

The project is built using a modern architecture with a React Native frontend, FastAPI backend, and MySQL database.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/dashflinch/Urbancart-Frontend.git
```

```
cd Urbancart-Frontend
```

Install Dependencies

```bash
npm install
```

Start Metro

```bash
npm start
```

Run Android

```bash
npm run android
```

Run iOS

```bash
npm run ios
```

---

# ⚙ Backend

The backend is built using FastAPI.

Install requirements

```bash
pip install -r requirements.txt
```

Run Server

```bash
uvicorn app.main:app --reload
```
